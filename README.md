# java-array-programs
Overview:
This repository contains basic Java programs that demonstrate array operations. The programs include reversing an array and counting vowels in an array. The code was written in Java and stored in GitHub. Continuous Integration and Continuous Deployment were implemented using Jenkins to automatically build the project.
Programs Included:
Reverse Array Program:
Reverses the elements of an array.
Demonstrates array traversal and looping concepts in Java.
Vowel Count in Array:
Counts the number of vowels present in a character array.
Demonstrates conditional statements and array processing.
